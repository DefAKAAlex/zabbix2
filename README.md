Изначально, я создал Item для отображения информацию об загрузке CPU используя **key:system.cpu.util**

При прикрепление к хосту шаблона своего и Linux by Zabbix Agent столкнулся с проблемой - оба шаблона используют один ключ

![Проблема](https://github.com/DefAKAAlex/zabbix2/blob/main/img/task23-1.png)

Решил создать свой параметр для отображения информации о загрузке CPU


Создаю свой параметр, в файле  */etc/zabbix/zabbix_agentd.conf*
```
UserParameter=my.cpu,vmstat 1 2 | tail -1 | awk '{print 100 - $15}'
```

использую утилиту **vmstat** чтобы получить данные о простое процессора и возвращаю разницу между этим значением и 100%

Такой вид приобретает мой новый шаблон
